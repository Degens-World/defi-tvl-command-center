# DeFi TVL Command Center

Real-time Total Value Locked (TVL) dashboard across all DeFi chains and protocols, powered by the [DeFiLlama](https://defillama.com) public API.

## Features

- **Chain TVL Breakdown** — doughnut chart + horizontal bar chart of top 15 chains by TVL
- **Protocol Explorer** — paginated table of 1000+ protocols with category/chain filters, search, and sort
- **Top Movers** — biggest 7D TVL gainers and losers (filtered to protocols with >$1M TVL)
- **DeFi TVL History** — line chart of total DeFi TVL with 1M / 3M / 6M / 1Y range selector
- **Auto-refresh** every 5 minutes; manual refresh button

## Tech Stack

- Vanilla JS + Chart.js 4.x
- DeFiLlama REST API (no API key required)
- Fully static — deploys anywhere
